# THEGO
Python-based TCP/UDP port scanner with protocol-aware service detection and binary protocol fingerprinting.

FEATURES
- TCP scanning
- UDP scanning
- Banner grabbing
- DNS detection
- MySQL fingerprinting
- PostgreSQL probing
